# PC-Components-Inventory-Manager
A simple inventory management app for Computer parts that is made in Express, with Love, by Chheang, for TOP.

## Livesite
You should be able to [check the live site here](https://pc-components-inventory-manager-production.up.railway.app/catalog)
## Languages and Tech used
  - JS
  - Express
  - PUG
  - MongoDB
  - Mongoose

## Dependencies used
  - Multer
  - Helmet
  - Compression
  - Async
  - Mongoose
  - Morgan
